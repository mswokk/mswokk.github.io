자사 임베디드 기기와 통신할 수 있는 Java client library
  • 개발 기여도 100%
  • Java, JVM, Android
  • RxJava, RS232, BT

  • Rx의 장점을 최대한 활용했던 프로젝트

기존 버전의 Client library는 API가 Callback 형식으로 설계되어 있어서 API 마다 listener 가 별도로 선언됐습니다. 그리고 Blocking 방식의 호출을 하려면 추가적인 구현이 필요했습니다. RxJava 를 이용해 이런 문제를 해결하고 코드길이를 1/4로 줄였습니다. Serial 통신 구현시 Protocol 요구사항에 의해 Thread-safe한 전이중 통신이 가능해야 했는데 이 또한 Rx를 이용해 수월하게 해결했습니다.

  • 시키지 않아도 능동적으로...

기존 Library의 문제를 대폭 보완하고자 스스로 시간을 내서 2주만에 Prototyping을 마쳤고 나중에 실제 의료기기 제품에 탑재됐습니다.
